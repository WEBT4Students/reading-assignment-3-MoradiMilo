Web Server

A Web Server is a computer software abd underlying hardware that accepts requests via HTTP or its secure variant HTTPS

A web server connects to the internet and supports physical data interchange woth other devices connected to the web. On the software side, a web server included several parts that control how web users acces hosted files

They store data on the server using some kind of database (server-side storage) then run server-side code to retieve needed data, insert it into static page templates, and serve the resulting HTML to the client to be displayed by the users browser

Page content refers to all the information contained in a website. Page content can be displayed as text, links, images, audio, animation or videos among other things

Web Client

A web client is an application that communicates with a web server, using Hypertext Transfer Protocol (HTTP)

The purpose of a web browser is to fetch content from the Web or from a local storage device and display it on a user's device.

Declarations for font families 
 he font-family property specifies the font for an element.

        The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font.
        
        two types of font family
        
        family-name - The name of a font-family, like "times", "courier", "arial", etc.
        
        generic-family - The name of a generic-family, like "serif", "sans-serif", "cursive", "fantasy"

Declarations for font sizes 
 Declarations for font sizes

        The font-size property sets the size of the text.
        
        Absolute size:
        
        Sets the text to a specified size
        
        Does not allow a user to change the text size in all browsers (bad for accessibility reasons)
        
        Absolute size is useful when the physical size of the output is known
        
        Relative size:
        
        Sets the size relative to surrounding elements
        
        Allows a user to change the text size in browsers
        
        Set Font Size With Pixels => font-size: 40 px;

        Set Font Size With Em => font-size: 2.5em; (40 px / 16 = 2.5 em)
        
        Use Combination Of Percent and EM => he solution that works in all browsers, is to set a default font-size in percent for the element
